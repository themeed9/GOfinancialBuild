# Skill: Component Builder

---

## Purpose

Create reusable, accessible UI components.

---

## Rules

- Use CSS Modules
- Use tokens only
- No inline styles

---

## Structure

Component/
- Component.tsx
- Component.module.css

---

## Requirements

- Accessible (aria labels)
- Touch target ≥ 44px
- Clear props

---

## Output

Reusable, clean, minimal component
